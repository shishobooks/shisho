# Supported Formats

Shisho supports a variety of book formats across three media types.

## Ebooks

- **EPUB** — Full metadata extraction including title, authors, series, description, cover art, and more
- **PDF** — Basic metadata support

## Audiobooks

- **M4B** — Full metadata extraction including title, authors, narrators, series, chapters, and cover art

## Comics

- **CBZ** — Full metadata extraction from ComicInfo.xml including title, authors, series, and cover art

## Downloads

Shisho can generate download files in additional formats:

- **KePub** — Kobo-optimized EPUB format for Kobo e-readers
